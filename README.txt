1. - PROPOSTA   

    Construir uma biblioteca para a criação e uso de redes neurais, com o objetivo de usá-la para problemas de classificação de imagens de animais.
    Em específico, o problema objetivo é a classificação de fotografias de cães e gatos obtidas em https://www.kaggle.com/datasets/tongpython/cat-and-dog.

    A intenção é estudar técnicas e arquiteturas de convolutional neural networks(CNN) e deep neural networks(DNN), assim como observar fenômenos como a extração
    de caracterpisticas de imagens nas camadas da rede neural (encoding)

    Técnicas à implementar: kernel convolution, deep networks, maxpooling, multidimensional node input/tensors, varied activation funtions

    objetivos: classificação do dataset apresentado com razoável precisão, implementação de ferramentas visuais para observação da rede neural em funcionamento, comparação de arquiteturas

    bibliotecas: numpy, pygame(visualization), Pillow(image manipulation), matplotlib (image manipulation)

    repositório: https://github.com/Guilherme-jose/ml
